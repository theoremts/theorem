# theoremts-cli

CLI for [Theorem](https://github.com/theoremts/theorem) — formal verification for TypeScript.

```bash
npm install -D theoremts-cli

theorem verify src/     # prove contracts
theorem scan src/       # detect risks (alpha)
theorem suggest src/    # auto-generate contracts (alpha)
```

See [full documentation](https://github.com/theoremts/theorem).
